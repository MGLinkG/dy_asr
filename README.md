# 抖音自动续火花 (Douyin Streak Auto)

这是一款基于 Electron + React + Playwright 的跨平台桌面应用程序，支持 Windows 和 Mac。它可以全自动在后台运行，定时给指定的抖音好友发送文本消息或视频，实现“续火花”功能。

## 主要功能

1. **跨平台支持**：支持 Windows 和 Mac
2. **好友列表获取**：一键获取抖音好友列表及头像
3. **定时执行**：可配置 Cron 表达式实现每天定时运行续火花脚本
4. **手动/定时双模式**：支持手动执行或定时在后台静默执行
5. **灵活的好友选择**：可以在列表中勾选指定好友执行续火花
6. **丰富的消息类型**：支持发送纯文本或指定本地视频文件

## 开发与构建

### 1. 环境准备

- Node.js (>= 18)
- 本机需安装 Google Chrome 或 Microsoft Edge 浏览器

### 2. 安装依赖

```bash
npm install
```

### 3. 启动开发环境

```bash
npm run dev
```

### 4. 打包构建

打包 Windows 版本 (.exe):

```bash
npm run build:win
```

打包 Mac 版本 (.dmg):

```bash
npm run build:mac
```

> **注意**: 如果在安装或构建过程中遇到证书问题，可以尝试设置 `export NODE_TLS_REJECT_UNAUTHORIZED=0`。

## 使用说明

1. 首次使用时，点击 **扫码登录**。会弹出一个浏览器窗口，请使用抖音 APP 扫码登录，登录成功后关闭窗口，状态会自动保存。
2. 点击 **获取好友**，列表会展示你的抖音好友，支持头像显示。
3. 在左侧配置面板中，设置 **定时任务** (Cron 表达式)、**消息类型** (文本/视频) 以及相应的内容。
4. 在右侧好友列表中，**勾选** 你想要续火花的好友。
5. 点击 **手动执行** 可以立即测试脚本效果。
6. 配置完成后，只要保持本程序在后台运行，它就会按照 Cron 表达式定时静默执行。

## 注意事项

- 请合理设置发送频率与文本内容，过度频繁的自动化操作可能会被抖音判定为异常并限制私信功能。
- 本程序基于 Playwright 自动化操作网页版抖音，若抖音网页版 DOM 结构发生大幅改变，可能需要更新 `src/main/automation.ts` 中的 DOM 选择器。
